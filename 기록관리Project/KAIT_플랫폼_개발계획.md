# 감어인 Cloud — 기록관리 SaaS 플랫폼 개발 계획

> 작성일: 2026-03-10 | 기반: KAIT 프로세스 개선 제안서

---

## 1. 플랫폼 비전

> "기록관리의 모든 것을 클라우드 하나로 — AI가 도와주는 기록관리 SaaS"

**감어인 Cloud**는 기존 설치형 감어인 5종 솔루션을 클라우드 단일 플랫폼으로 통합하고,
AI 자동화·지식베이스·공개 아카이브 포털을 하나의 SaaS로 제공한다.

---

## 2. 핵심 모듈 구성

| 모듈 | 설명 | 대응 제안 |
|------|------|-----------|
| **대시보드** | 기관별 기록 현황, AI 처리 현황, 알림 | 전체 |
| **기록 관리** | 수집·등록·분류·보존·검색 (감어인R+M 통합) | 제안 1 |
| **AI 자동화** | OCR·메타데이터 추출·구술 전사·요약 | 제안 1, 2 |
| **지식베이스** | 컨설팅 사례 검색, RAG AI 어시스턴트 | 제안 3 |
| **아카이브 포털** | 대국민 공개 서비스 (타임라인·전시·검색) | 제안 5 |
| **서고 모니터링** | IoT 환경 센서 대시보드 | 제안 6 |
| **관리자** | 테넌트 관리, 사용자 권한, 요금제 | SaaS 인프라 |

---

## 3. 타겟 고객 및 요금제

| 구분 | 대상 | 요금제 |
|------|------|--------|
| Starter | 대학·재단·종교기관 | 월 30만원 (기본 모듈) |
| Professional | 공공기관·지방공기업 | 월 150만원 + AI 추가 |
| Enterprise | 중앙행정기관·대기업 | 커스텀 (온프레미스 옵션) |

---

## 4. 기술 스택

| 레이어 | 기술 |
|--------|------|
| Frontend | React + TypeScript + Tailwind CSS |
| Backend | FastAPI (Python) |
| DB | PostgreSQL + pgvector (RAG용) |
| AI | OpenAI API / Whisper STT / OCR |
| 인프라 | AWS (또는 NCP) + CSAP 인증 추진 |
| 배포 | Docker + Kubernetes |

---

## 5. 개발 로드맵

### Phase 1 — MVP (6개월)
- 대시보드, 기록 등록·검색, AI 메타데이터 추출
- 기본 SaaS 멀티 테넌트 구조
- Starter 요금제 론칭

### Phase 2 — 확장 (6~18개월)
- 구술 AI 전사 모듈
- 지식베이스 RAG 어시스턴트
- 아카이브 공개 포털 템플릿

### Phase 3 — 고도화 (18개월~)
- IoT 서고 모니터링 연동
- CSAP 인증 획득 (공공 SaaS)
- Enterprise 온프레미스 옵션

---

## 6. 프로토타입 범위 (현재)

단일 HTML 파일로 주요 5개 화면 UI 프로토타입 구현:

1. **대시보드** — 통계 카드, AI 처리 현황, 최근 등록 기록
2. **기록 관리** — 기록 목록, 검색 필터, 상세 패널
3. **AI 자동화** — 파일 업로드, OCR 결과, 전사 결과
4. **지식베이스** — 검색창, 사례 카드, AI 답변
5. **아카이브 포털** — 공개 타임라인 미리보기
