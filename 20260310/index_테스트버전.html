실용 Tips
실용적인 팁 10가지
📋 Tip 1: AI 작업시킬 때 먼저 하는 것들
AI에게 바로 실행시키지 마라! 먼저 생각하게 하라.
① "검토 해봐" — 현재 상태를 먼저 파악시키기
② "계획 세워봐" — 실행 전에 로드맵을 만들게 하기
③ "방안을 제시해봐" — 여러 대안을 먼저 보기
→ 실행은 마지막! 검토 → 계획 → 방안 → 그다음 실행
🖥️ Tip 2: Claude 앱/데스크탑 vs Claude Code
Claude 앱/데스크탑 = 컨설턴트
상담자

• 전체 구조를 보고 "왜?" 되물음
• "잠깐, 그거 왜 MCP로 감싸요?"
  라고 물음
• 실행 도구가 없으니
  대신 생각을 더 하게 됨
Claude Code = 현장 기술자
실행자

• 코드 쓰고 파일 만들고 명령 실행
• "MCP 연결하라고? 알겠습니다!"
  하고 바로 코드 짬
• 실행 도구(터미널)가 있으니
  바로 행동하려 함
💡 Claude 앱/데스크탑에서 전략·설계 → Claude Code에서 실행
</> Tip 3: Vanilla vs React 비교
Vanilla
순수 HTML/CSS/JS · 프레임워크 없음
📁 api/ · pages/ · assets/ · scripts/
📄 index.html · 404.html

파일 만들면 바로 실행
빌드 과정 불필요 · 학습 곡선 보통
UI 재사용이 번거움
React/Next.js
컴포넌트 기반 개발 · 라이브러리 필요
📁 app/ · components/ · public/ · lib/

UI 컴포넌트 재사용 쉬움
대규모 웹사이트에 유리
학습이 필요함 · 빌드 과정 필수
